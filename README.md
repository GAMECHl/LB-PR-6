### Висновок
#### Порівняння швидкості методів: Метод Гауса виявився найшвидшим алгоритмом з лінійною складністю O(n), його час виконання не перевищує мілісекунди навіть для полігонів з 1000 вершин. Метод Монте-Карло значно повільніший – при 10000 точок час виконання становить 0.1-0.2 секунди, що в сотні разів довше. Для практичних задач метод Гауса є оптимальним вибором завдяки поєднанню абсолютної точності та високої швидкодії.
#### Точність Монте-Карло: Дослідження показало, що для прийнятної похибки < 1% достатньо 10000-20000 ітерацій. При 1000 точок похибка становить 2-5%, при 10000 точок – 0.5-2%, а при 50000+ точок – менше 0.5%. Метод доцільний лише для освітніх цілей або складних фігур, де аналітичний розрахунок неможливий, оскільки поступається методу Гауса як за точністю, так і за швидкодією.
